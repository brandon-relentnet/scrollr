🏥 Health & Status Endpoints
GET /api/accounts/health

Comprehensive accounts service health check with database connectivity, JWT configuration, uptime metrics, and memory usage

Status States: healthy | degraded | unhealthy
Dependencies: PostgreSQL database, JWT secret validation
Returns: service info, uptime, memory usage, environment details
GET /api/finance/health

Advanced finance service health with database, Finnhub WebSocket status, cache metrics, and real-time client monitoring

Status States: healthy | degraded | unhealthy
Dependencies: PostgreSQL database, Finnhub API, WebSocket connections
Returns: client count, cache metrics, trade data age, connection status
GET /api/sports/health

Enhanced sports service health with database, ESPN API accessibility, and WebSocket client tracking

Status States: healthy | degraded | unhealthy
Dependencies: PostgreSQL database, ESPN API access
Returns: client count, API accessibility status, service metrics

Code Examples

JavaScript/TypeScript 
// Authentication
const response = await fetch('https://scrollr.olvyx.com/api/accounts/auth/login', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    identifier: 'user@example.com',
    password: 'yourpassword'
  })
});

const { token } = await response.json();

// WebSocket Connection
const ws = new WebSocket('wss://scrollr.olvyx.com/api/finance/ws');

ws.onopen = () => {
  // Send connection message
  ws.send(JSON.stringify({
    type: 'connection',
    timestamp: Date.now()
  }));
  
  // Request filtered data
  ws.send(JSON.stringify({
    type: 'filter_request',
    filters: ['symbol_AAPL', 'symbol_GOOGL']
  }));
};

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  console.log('Received:', data);
};
Python 
import requests
import websocket
import json

# Authentication
response = requests.post('https://scrollr.olvyx.com/api/accounts/auth/login', 
    json={
        'identifier': 'user@example.com',
        'password': 'yourpassword'
    }
)

token = response.json()['token']

# WebSocket Connection
def on_open(ws):
    # Send connection message
    ws.send(json.dumps({
        'type': 'connection',
        'timestamp': int(time.time() * 1000)
    }))
    
    # Request filtered data
    ws.send(json.dumps({
        'type': 'filter_request',
        'filters': ['symbol_AAPL', 'symbol_GOOGL']
    }))

def on_message(ws, message):
    data = json.loads(message)
    print(f'Received: {data}')

ws = websocket.WebSocketApp('wss://scrollr.olvyx.com/api/finance/ws',
    on_open=on_open,
    on_message=on_message)

ws.run_forever()
